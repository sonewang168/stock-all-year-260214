## 📋 系統概覽

純前端 HTML/JS 投資追蹤工具，所有資料存在瀏覽器 `localStorage`（key: `bp2026v5`），無後端、無登入、開即用。

### 8 個功能頁面

| 頁面 | 功能 |
|------|------|
| 📋 作戰 | 13 個月任務清單 + 核心原則打勾 |
| 💼 持股 | 新增/編輯/刪除持股（張＆零股） |
| 📝 週紀錄 | 週一/週五/封關前快照，自動算損益 |
| 🌍 國際 | 美光、輝達、SK海力士、費半追蹤 |
| 💰 月結算 | 月底持股估值 → 自動算報酬率 → 匯出給 Claude |
| 💎 了結 | 獲利/停損紀錄，自動扣減持股 |
| 📊 半年報 | H1(2-7月) / H2(8-隔年2月) 自動彙整 |
| 🎯 目標 | 各股目標價 + 潛在報酬計算 |

## 🔄 操作流程

### 每週例行（花 5 分鐘）

```
📝 週紀錄 → ＋新增
  → 選日期 + 週一/週五
  → 輸入每檔現價（損益自動算）
  → 填國際數據（美光、費半等）
  → 備忘錄
  → 💾 儲存
```

### 封關前（農曆年前最後交易日）

```
📝 週紀錄 → ＋新增
  → 類型選 🧧封關前
  → 輸入每檔現價
  → 每檔選擇決策：續抱 / 了結 / 觀察 / 加碼 / 減碼
  → 備忘錄寫下過年期間觀察重點
  → 💾 儲存
```

### 賣股時

```
💎 了結 → ＋新增了結
  → 選持股 → 輸入賣出價（損益自動算）
  → 確認 → 持股自動扣減
```

### 每月底

```
💰 月結算 → ＋新增
  → 選月份 → 輸入各股月底價（全部自動算）
  → 填配息、交易次數、自評等級
  → 💾 儲存
  → 📤 匯出 → 📋 複製 → 貼到 Claude 對話 → 請 Claude 評核
```

### 半年結算（第 6、13 個月）

```
📊 半年報
  → 自動彙整所有月結算
  → 📤 匯出 H1/H2 報表
  → 貼給 Claude 做半年度評核
```

## ⭐ 自評等級標準

| 等級 | 說明 | 核心判斷 |
|------|------|----------|
| **S** | 完美執行 | 紀律 100%、正報酬、零衝動交易 |
| **A** | 表現優良 | 紀律佳、獲利或持平、≤1 次換股 |
| **B** | 及格過關 | 小虧可控、偶爾手癢但沒亂動 |
| **C** | 需要檢討 | 違規 1-2 次、衝動交易、換股 >1 次 |
| **D** | 嚴重失控 | 多次違規、大幅虧損、情緒化操作 |

> **核心原則：紀律 > 報酬**

## 🛡️ 核心原則（六不原則）

1. ❌ 不追漲停，等拉回再接
2. ❌ 不在虧損狀態下換股
3. ❌ 每月換股 ≤ 1 次
4. ❌ 定存不動，除非崩盤加碼
5. ❌ 看盤：每週 1-2 次
6. ✅ 用賺到的錢開新倉
